Матеріали для покращення (Апгрейд)

Сіра Утоплена Нагальна

Для рівнів: 1 — 3

Залишки мерців, що викинуло на берег. Використовується ковалями для базового покращення звичайної зброї та щитів.

Ресурс: Мечі / Топори / Копья

Волосина Відьми

Для рівнів: 4 — 6

Рідкісний магічний матеріал, наповнений чистою сіллю. Необхідний для прокачування посохів, жезлів та вищих магічних клинків.

Ресурс: Посохи / Жезли

Мертва Сіль

Для рівнів: 7 — 9

Кристалізована есенція найсильніших істот острова. Максимально збільшує базові показники шкоди будь-якого важкого озброєння.

Ресурс: Великі мечі / Молоти
